# slimr

A package for parsing, processing and analyzing [SLiM](https://messerlab.org/slim/) output files in R.

## Installation

You need to install `devtools` first:

```
install.packages("devtools")
devtools::install_github("bodkan/slimr")

library(slimr)
```
